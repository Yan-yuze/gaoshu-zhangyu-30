## (1) 信道的定义、分类及数学模型；信道中的噪声

### 一、信道的分类体系

```
信道
├── 狭义信道（物理传输媒质）
│   ├── 无线信道：地波、天波、视线传播
│   └── 有线信道：明线、对称电缆、同轴电缆、光纤
│
└── 广义信道（从信号传输角度划分）
    ├── 调制信道（研究调制/解调）
    │   ├── 恒参信道
    │   └── 随参信道
    └── 编码信道（研究编码/译码，用转移概率描述）
        ├── 无记忆信道
        └── 有记忆信道
```

### 二、无线信道（三种传播方式）

| 传播方式 | 频率范围 | 传播距离 | 典型应用 | 稳定性 |
|----------|----------|----------|----------|--------|
| **地波** | 30kHz~3MHz | 数百~数千km | AM广播、航海 | 高 |
| **天波** | 3~30MHz | 数千~1万km | 短波广播 | 低（受电离层影响） |
| **视线波** | >30MHz | 数十km | 电视、手机、卫星 | 高（需无障碍） |

### 三、有线信道

| 类型 | 特点 | 用途 |
|------|------|------|
| 明线 | 损耗低，易受天气影响，噪声敏感 | 早期电话 |
| 对称电缆（双绞线） | 损耗比明线大，性能稳定 | 电话用户接入 |
| 同轴电缆 | 外导体接地屏蔽 | 有线电视 |
| 光纤 | 全反射传输，损耗极小 | 长距离通信 |

**光纤原理**：纤芯折射率 > 包层折射率 → 全反射

### 四、信道数学模型

**(a) 调制信道模型**

$$e_o(t) = k(t) \cdot e_i(t) + n(t)$$

- k(t)：乘性干扰（信道特性）
  - 基本不变 → **恒参信道**
  - 随机变化 → **随参信道**
- n(t)：加性噪声

**(b) 恒参信道模型**
- 等效为线性时不变网络
- 用幅频特性和相频特性描述
- 无失真条件：幅频特性为常数，相频特性为通过原点的直线

**(c) 编码信道模型**
- 用转移概率描述
- 二进制对称信道转移矩阵：

$$P = \begin{bmatrix} 1-p & p \\ p & 1-p \end{bmatrix}$$

### 五、信道中的噪声

**按来源分类：**

| 分类 | 产生原因 | 举例 |
|------|----------|------|
| 人为噪声 | 人类活动 | 电火花、汽车点火、荧光灯 |
| 自然噪声 | 自然界电磁辐射 | 闪电、宇宙噪声、热噪声 |

**按性质分类：**

| 分类 | 特点 |
|------|------|
| 脉冲噪声 | 突发、振幅大、频谱宽 |
| 窄带噪声 | 连续已调正弦波，频率可测 |
| 起伏噪声 | 遍布时域和频域的随机噪声（热噪声、散弹噪声等） |

**高斯白噪声**：瞬时值服从正态分布，功率谱密度为常数 $P_n(f) = n_0/2$

## (2) 恒参信道、随参信道及其对信号的影响

### 一、恒参信道的影响

**(a) 频率失真（幅度失真）——线性失真**
- 幅频特性不理想导致
- 可通过线性网络补偿

**(b) 相位失真——线性失真**
- 群延迟特性不理想导致
- 波形形状发生畸变

**(c) 其他失真**
- 非线性失真：产生新的谐波分量，难以消除
- 频率偏移
- 相位抖动

### 二、随参信道的特点
1. 传输衰减随时间随机变化
2. 传输时延随时间随机变化
3. **多径传播**：多条路径到达，各路径衰减和时延均变化

### 三、多径效应（重点）

**两径传播模型分析：**

传输函数幅度特性：
$$|H(\omega)| = 2k\left|\cos\frac{\omega\Delta\tau}{2}\right|$$

- 传播极点（最强）：$f = n/\Delta\tau$
- 传播零点（最弱）：$f = (n+1/2)/\Delta\tau$

**相关带宽**：
$$B_c = \frac{1}{\Delta\tau}$$

> 工程设计中，信号带宽应为相关带宽的 1/5 ~ 1/3

**多径效应结论：**
1. 单一频率信号变为包络和相位受调制的窄带信号（衰落信号）
2. 多径传播引起频率弥散（单一谱线→窄带频谱）

## (3) 信道容量的概念、计算及香农公式

### 一、信道容量定义

信道中能够传递的最大平均信息速率。

离散信道：$C = \max_{P(x)} I(X,Y)$ （bit/符号）

连续信道：$C_t = R_s \cdot C$ （bit/s）

### 二、香农公式（重点★★★）

$$C = B \log_2\left(1 + \frac{S}{N}\right)$$

或

$$C = B \log_2\left(1 + \frac{S}{n_0 B}\right)$$

- C：信道容量（bit/s）
- B：信道带宽（Hz）
- S：信号平均功率（W）
- N：噪声功率（W），N = n₀B
- n₀：噪声单边功率谱密度（W/Hz）

### 三、香农公式的推论

1. **增大信号功率S** → 可增加C，$S \to \infty$ 时 $C \to \infty$
2. **减小噪声功率N（或n₀）** → 可增加C，$N \to 0$ 时 $C \to \infty$
3. **增大带宽B** → 可增加C，但 $\lim_{B\to\infty} C = 1.44 S/n_0$（有上限）
4. **C一定时，B、S/N、t三者可互换**（带宽换信噪比、带宽换时间）

### 四、平均互信息量与信道容量

平均互信息量：
$$I(X,Y) = H(X) - H(X/Y) = H(Y) - H(Y/X)$$

对称信道容量简化：匹配信源为等概分布，用方法2（$I(X,Y) = H(Y) - H(Y/X)$）计算量更小。

## 必背公式速查

| 公式 | 含义 |
|------|------|
| $e_o(t) = k(t) \cdot e_i(t) + n(t)$ | 调制信道模型 |
| $B_c = 1/\Delta\tau$ | 多径相关带宽 |
| $C = B\log_2(1+S/N)$ | 香农公式 |
| $C = B\log_2(1+S/n_0B)$ | 香农公式（n₀形式） |
| $\lim_{B\to\infty} C = 1.44 S/n_0$ | 带宽无穷大时信道容量极限 |
| $I(X,Y) = H(Y) - H(Y/X)$ | 平均互信息量 |

## 常见易错点

1. 恒参信道≠无失真，只是特性不随时间变化
2. 多径效应使信号频率选择性衰落，相关带宽是关键参数
3. 香农公式中B趋近无穷时C有上限，不是无限增大
4. 调制信道模型中的k(t)是乘性干扰，n(t)是加性干扰
