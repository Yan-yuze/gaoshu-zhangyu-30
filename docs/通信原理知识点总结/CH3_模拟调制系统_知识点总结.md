## (1) 线性调制、非线性调制、相干解调、非相干解调

### 一、调制的概念与分类

**调制**：把低频基带信号搬移到高频信道通带的过程  
**解调**：从已调信号恢复基带信号的反过程

**调制目的**：适合信道传输、有效辐射、实现频率分配、多路复用、提高抗噪性

| 分类依据 | 类别 |
|----------|------|
| 调制信号m(t) | 模拟调制 / 数字调制 |
| 载波c(t) | 连续波调制 / 脉冲波调制 |
| 载波参数 | 幅度调制 / 频率调制 / 相位调制 |
| H(ω)特性 | **线性调制**（AM/DSB/SSB/VSB）/ **非线性调制**（FM/PM） |

### 二、线性调制（幅度调制）总览

通用模型（**滤波法**）：
$$s(t) = [m(t)\cos\omega_c t] * h(t)$$

通用模型（**相移法**）：
$$s(t) = s_I(t)\cos\omega_c t + s_Q(t)\sin\omega_c t$$

### 三、相干解调 vs 非相干解调

| 解调方式 | 原理 | 适用 |
|----------|------|------|
| **相干解调** | 接收信号 × 本地载波 → 低通滤波 | 所有线性调制 |
| **非相干解调（包络检波）** | 提取包络 → 低通滤波 | AM（需大信噪比） |

> 相干载波必须与发送载波同频同相，否则失真

## (2) 各类模拟调制系统详解

### 一、AM（调幅）★★★

**时域**：$s_{AM}(t) = [A_0 + m(t)]\cos\omega_c t$  
**要求**：$A_0 + m(t) \geq 0$（不发生过调幅）

**频域**：
$$S_{AM}(\omega) = \pi A_0[\delta(\omega+\omega_c) + \delta(\omega-\omega_c)] + \frac{1}{2}[M(\omega+\omega_c) + M(\omega-\omega_c)]$$

**带宽**：$B_{AM} = 2f_m$

**功率**：$P_{AM} = \frac{A_0^2}{2} + \frac{\overline{m^2(t)}}{2} = P_c + P_s$

**调制效率**：$\eta_{AM} = \frac{P_s}{P_{AM}} = \frac{\overline{m^2(t)}}{A_0^2 + \overline{m^2(t)}}$

- 单音100%调制时：$\eta_{AM} = 1/3 \approx 33.3\%$
- 功率利用率低！

**解调**：相干解调或包络检波

### 二、DSB-SC（双边带抑制载波）

**时域**：$s_{DSB}(t) = m(t)\cos\omega_c t$

**带宽**：$B_{DSB} = 2f_m$

**调制效率**：$\eta_{DSB} = 100\%$（无载波分量）

**解调**：必须相干解调

### 三、SSB（单边带）

**时域**（相移法）：
$$s_{SSB}(t) = \frac{1}{2}m(t)\cos\omega_c t \mp \frac{1}{2}\hat{m}(t)\sin\omega_c t$$

其中 $\hat{m}(t)$ 是 $m(t)$ 的**希尔伯特变换**

**带宽**：$B_{SSB} = f_m$（是AM/DSB的一半！）

**功率**：$P_{SSB} = \frac{1}{4}\overline{m^2(t)}$（是DSB的1/4）

**优点**：节省带宽和功率  
**难点**：滤波法在载频处难实现陡峭截止；含直流低频时不适用

**希尔伯特变换**：$H_h(\omega) = -j \cdot \text{sgn}(\omega)$（宽带90°移相网络）

### 四、VSB（残留边带）

介于DSB和SSB之间，滤波器在载频处有互补对称特性  
**要求**：$H_{VSB}(\omega-\omega_c) + H_{VSB}(\omega+\omega_c) = \text{常数}$

$P_{SSB} < P_{VSB} < P_{DSB}$  
$B_{SSB} < B_{VSB} < B_{DSB}$

## (3) 非线性调制（角度调制）

### 一、角度调制基本概念

$$s_m(t) = A\cos[\omega_c t + \varphi(t)]$$

**(a) 相位调制（PM）**：$\varphi(t) = k_p m(t)$

**(b) 频率调制（FM）**：$\frac{d\varphi(t)}{dt} = k_f m(t)$，即 $\varphi(t) = k_f \int m(t)dt$

| 参数 | PM | FM |
|------|-----|-----|
| 调制指数 | $m_p = k_p A_m$ | $m_f = k_f A_m/\omega_m = \Delta f/f_m$ |
| 最大频偏 | — | $\Delta f = k_f A_m$ |

### 二、窄带调频（NBFM）

**条件**：$|\varphi(t)|_{max} \ll \pi/6$（约0.5）

**带宽**：$B_{NBFM} = 2f_m$（与AM相同）

### 三、宽带调频（WBFM）★★★

**单音调频表达式**：
$$s_{FM}(t) = A\sum_{n=-\infty}^{\infty} J_n(m_f)\cos[(\omega_c + n\omega_m)t]$$

- $J_n(m_f)$ 为第一类n阶贝塞尔函数
- 理论上频谱无限宽

**卡森公式（带宽）**：
$$B_{WBFM} \approx 2(m_f + 1)f_m = 2(\Delta f + f_m)$$

- $m_f \ll 1$：$B \approx 2f_m$（窄带）
- $m_f \gg 1$：$B \approx 2\Delta f$（宽带，带宽由最大频偏决定）

### 四、调频信号产生与解调

**产生**：直接调频（VCO）、间接调频（阿姆斯特朗法）

**解调**：非相干（鉴频器 = 微分 + 包络检波）

## (4) 模拟调制系统抗噪声性能

### 一、分析模型

$$G = \frac{S_o/N_o}{S_i/N_i}$$

G越大，抗噪声性能越好。

### 二、各系统制度增益与性能对比

**条件**：相同$S_i$、$n_0$、$f_m$，AM按100%调制

| 系统 | $S_o/N_o$ | G | 带宽 |
|------|-----------|-----|------|
| AM | $\frac{\overline{m^2(t)}}{n_0 f_m}$ | $\frac{2\overline{m^2(t)}}{A_0^2+\overline{m^2(t)}} \leq 2/3$ | $2f_m$ |
| DSB | $\frac{\overline{m^2(t)}}{n_0 f_m}$ | 2 | $2f_m$ |
| SSB | $\frac{\overline{m^2(t)}}{n_0 f_m}$ | 1 | $f_m$ |
| VSB | $\approx \frac{\overline{m^2(t)}}{n_0 f_m}$ | ≈1 | $(1\sim2)f_m$ |
| FM(WB) | $\frac{3}{2}m_f^2\frac{S_i}{n_0 f_m}$ | $3m_f^3$（大信噪比） | $2(m_f+1)f_m$ |

> **关键结论**：DSB和SSB输出信噪比相同（但$G_{DSB}=2, G_{SSB}=1$）  
> WBFM通过带宽换取高输出信噪比！

### 三、门限效应

- AM包络检波和FM鉴频在小信噪比时出现
- 输出信噪比急剧恶化，信号被噪声淹没
- FM可通过锁相环解调降低门限值

## (5) 调制方式应用与频分复用

### 一、应用场景

| 调制方式 | 主要应用 |
|----------|----------|
| AM | 中短波无线电广播 |
| DSB | 应用较少 |
| SSB | 短波广播、语音频分复用、数据传输 |
| VSB | 电视广播、数据传输 |
| FM | 超短波通信、调频立体声广播 |

### 二、频分复用（FDM）

**原理**：多路信号用不同载频同时传输  
**复用信号带宽**：$B_{FDM} = n \cdot B_{SSB}$（n为路数）

**多路载波通话（SSB-FDM）**：
- 12路 → 基群
- 5个基群 → 超群（60路）
- 10个超群 → 主群（600路）
- 每路4kHz（含防护带）

## 必背公式速查

| 公式 | 含义 |
|------|------|
| $s_{AM}(t) = [A_0+m(t)]\cos\omega_c t$ | AM时域 |
| $s_{DSB}(t) = m(t)\cos\omega_c t$ | DSB时域 |
| $s_{SSB}(t) = \frac{1}{2}m(t)\cos\omega_c t \mp \frac{1}{2}\hat{m}(t)\sin\omega_c t$ | SSB时域 |
| $B_{AM}=B_{DSB}=2f_m$ | AM/DSB带宽 |
| $B_{SSB}=f_m$ | SSB带宽 |
| $B_{WBFM}=2(m_f+1)f_m=2(\Delta f+f_m)$ | 卡森公式 |
| $m_f = \Delta f / f_m$ | 调频指数 |
| $G_{DSB}=2, G_{SSB}=1$ | 制度增益 |
| $\eta_{AM} = \frac{\overline{m^2(t)}}{A_0^2+\overline{m^2(t)}}$ | AM调制效率 |

## 常见易错点

1. DSB和SSB输出信噪比相同但G不同（条件不同！）
2. AM调制效率低（最多50%），DSB=100%，SSB更省功率
3. 卡森公式记住 $2(\Delta f + f_m)$，窄带退化到 $2f_m$
4. 门限效应：AM包络检波和FM在小信噪比时出现
