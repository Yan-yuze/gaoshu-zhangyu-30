## (1) 通信系统模型、分类与通信方式

### 一、基本概念三要素

| 术语 | 定义 |
|------|------|
| 消息（message） | 信息的物理表现形式（语音、文字、图片等） |
| 信息（information） | 消息中包含的有效内容，是消息的内涵 |
| 信号（signal） | 消息的传输载体（如电信号） |

### 二、信号分类

- **模拟信号**：载荷消息的信号参量取值连续
- **数字信号**：载荷消息的信号参量只有有限个取值

### 三、通信系统一般模型（五大模块）

```
信源 → 发送设备 → 信道 → 接收设备 → 信宿
                ↑
              噪声源
```

| 模块 | 功能 |
|------|------|
| **信源** | 消息产生地，把消息转换成原始电信号（如摄像机、话筒） |
| **发送设备** | 将信源和信道匹配，进行变换（调制是最常见方式） |
| **信道** | 传输信号的物理媒质（无线信道/有线信道） |
| **接收设备** | 完成发送设备的反变换（解调、译码等） |
| **信宿** | 信息归宿点，将复原信号转换成消息（如喇叭、显示器） |
| **噪声源** | 各处噪声集中等效加到信道中 |

### 四、模拟通信系统模型

```
信源 → 调制器 → 信道 → 解调器 → 信宿
               ↑
             噪声源
```

- 调制器：频谱搬移，基带信号 → 频带信号
- 解调器：频带信号 → 基带信号

### 五、数字通信系统模型

```
信源 → 信源编码 → 信道编码 → 调制器 → 信道 → 解调器 → 信道译码 → 信源译码 → 信宿
                                      ↑
                                    噪声源
```

| 新增模块 | 功能 |
|----------|------|
| **信源编码** | 提高数字信号传输的有效性（模/数转换、压缩） |
| **信道编码** | 进行差错控制，提高通信系统可靠性 |
| 加密/解密 | 保密功能 |

### 六、数字通信系统的优缺点

**优点：**
1. 抗干扰能力强，可靠性高
2. 体积小，功耗低，易于集成
3. 便于进行各种数字信号处理
4. 有利于实现综合业务传输
5. 便于保密

**缺点：**
1. 占用频带宽
2. 对同步要求高，系统设备比较复杂

### 七、通信系统分类（6种分法）

| 分类依据 | 类别 |
|----------|------|
| 通信业务类型 | 电报、电话、数据、图像通信等 |
| 调制方式 | 基带传输 / 频带传输 |
| 信号特征 | 模拟通信系统 / 数字通信系统 |
| 传输媒质 | 有线 / 无线 |
| 工作波段 | 长波、中波、短波、远红外等 |
| 复用方式 | 频分复用(FDM)、时分复用(TDM)、码分复用(CDM) |

**常见调制方式：**

| 类型 | 调制方式 | 用途 |
|------|------|------|
| 线性调制 | AM（常规双边带） | 广播 |
| | DSB（双边带调幅） | 立体声广播 |
| | SSB（单边带调幅） | 载波通信、短波 |
| | VSB（残留边带） | 电视广播、传真 |
| 非线性调制 | FM（频率调制） | 微波中继、卫星通信、广播 |
| | PM（相位调制） | 中间调制方式 |
| 数字调制 | ASK（振幅键控） | 数据传输 |
| | FSK（频移键控） | 数据传输 |
| | PSK/DPSK（相移键控） | 数据传输 |
| | QAM、MSK等 | 数字微波、空间通信 |
| 脉冲调制 | PCM（脉码调制） | 市话、卫星、空间通信 |
| | ΔM（增量调制） | 军用、民用电话 |

### 八、通信方式

**按消息传送方向与时间：**

| 方式 | 含义 |
|------|------|
| 单工通信 | 消息只能单向传输（如广播） |
| 半双工通信 | 双方都可收发，但不能同时（如对讲机） |
| 全双工通信 | 双方可同时收发（如电话） |

**按码元排序：**

| 方式 | 含义 |
|------|------|
| 串行传输 | 码元一个接一个依次传输 |
| 并行传输 | 多个码元同时传输 |

## (2) 信息量的概念与计算

### 一、信息量定义

信息量 I 与消息出现概率 P 的关系：
- I 是 P 的函数
- P 越小，I 越大
- 独立消息的信息量可加

**核心公式：**

$$I = \log_a\frac{1}{P} = -\log_a P$$

| 底数 a | 单位 |
|--------|------|
| 2 | 比特（bit），可省略底数 |
| e | 奈特（nat） |
| 10 | 哈特莱（Hartley） |

**等概情况：**
$$I = \log_2 N \quad \text{（N为等可能结果数）}$$

> 例：16个等可能结果，$I = \log_2 16 = 4$ bit

### 二、平均信息量（信源熵）

**离散信源：**

$$H(X) = -\sum_{i=1}^{n} P(x_i) \log_2 P(x_i) \quad \text{(bit/符号)}$$

**连续信源：**

$$H(X) = -\int_{-\infty}^{+\infty} f(x) \log_2 f(x) \, dx$$

- 含义：每个符号所含信息量的统计平均值
- 等概时，$H(X) = \log_2 n$，此时熵最大
- 二进制信源：$H(\alpha) = -\alpha\log_2\alpha - (1-\alpha)\log_2(1-\alpha)$
- 当 $\alpha = 1/2$ 时熵最大（$H_{max} = 1$ bit/符号）

### 三、信息量与平均信息量的关系

- **单个符号法**：总信息量 $I = \sum n_i \cdot (-\log_2 P_i)$
- **平均信息量法**：总信息量 $I = N \cdot H(X)$
- 当消息序列足够长时，两种方法结果趋于一致
- 消息短时因频次不等于概率，存在误差

### 四、平均信息速率

$$R_b = R_B \times H(X)$$

即：平均信息速率 = 码元速率 × 平均信息量

## (3) 通信系统主要性能指标

### 一、两大核心指标

| | 有效性（速度） | 可靠性（质量） |
|------|------|------|
| **模拟系统** | 有效传输频带（带宽越小越好） | 输出信噪比（越高越好） |
| **数字系统** | 频带利用率、传输速率 | 误码率、误信率 |

> ⚠️ 有效性与可靠性是一对矛盾，可互换。

### 二、模拟通信系统性能指标

**(a) 有效性 — 有效传输频带**

- 传输同样的信号，所需带宽越小，频带利用率越高
- 不同调制方式占用带宽不同
- 信号带宽 B（单位：Hz）

**(b) 可靠性 — 输出信噪比**

- 用接收端最终输出信噪比（SNR）度量
- 不同调制方式在同样信道信噪比下解调后的信噪比不同
- 单位：dB

### 三、数字通信系统性能指标 — 有效性

**(a) 码元传输速率 R_B（传码率/符号速率/Baud率）**

$$R_B = \frac{1}{T_B} \quad \text{单位：波特（Baud）}$$

- 与进制数无关，只取决于码元宽度 T_B
- 例：1秒传2400个码元 → R_B = 2400B

**(b) 信息传输速率 R_b（传信率/比特率）**

$$R_b = R_B \times H(X) \quad \text{(b/s 或 bps)}$$

**等概M进制时：**

$$R_b = R_B \times \log_2 M$$

| 场景 | R_B (Baud) | R_b (b/s) |
|------|------------|------------|
| 等概二进制 | R_B | R_B × 1 = R_B |
| 等概四进制 | R_B | R_B × 2 |
| 等概八进制 | R_B | R_B × 3 |
| 等概M进制 | R_B | R_B × log₂M |

> 例：等概四进制，R_B=600Baud → R_b = 600×2 = 1200 b/s  
> 例：等概八进制，R_b=1800b/s → R_B = 1800÷3 = 600 Baud

**(c) 频带利用率**

$$\eta = \frac{R_B}{B} \quad \text{(B/Hz)}$$

$$\eta_b = \frac{R_b}{B} \quad \text{(b/(s·Hz))}$$

- 真正衡量传输效率的指标
- 相同速率下占用带宽越小，频带利用率越高

### 四、数字通信系统性能指标 — 可靠性

**(a) 误码率 P_e（码元差错率）**

$$P_e = \frac{\text{错误码元数}}{\text{传输总码元数}}$$

**(b) 误信率 P_b（信息差错率/误比特率）**

$$P_b = \frac{\text{错误比特数}}{\text{传输总比特数}}$$

**误码率与误信率的关系：**

$$P_e \leq P_b$$

- 二进制时：$P_e = P_b$（一个码元就是一个比特）
- 多进制时：$P_e \leq P_b$（一个错误码元可能包含多个错误比特）

> 例：4PSK传20个码元错2个 → P_e = 10%  
> 若每个错误码元有1个比特错 → P_b = 2/40 = 5%  
> 若每个错误码元有2个比特错 → P_b = 4/40 = 10%

## (4) 通信工程建设环保法律法规

> 来源：通信工程设计建设中基本的环保法律法规.docx

### 一、核心规定（10条要点）

1. **电池环保**：通信终端产品应使用氢镍电池、锂离子电池等可充电电池替代镉镍电池；不得使用汞含量大于0.0001%的锌锰及碱性锌锰电池和糊式电池等一次性电池。

2. **杀虫剂禁令**：通信工程中严禁使用持久性有机污染物做杀虫剂。

3. **电磁辐射危害**：电磁辐射对人体的危害主要表现在对人体神经系统的不良作用，其主要症状是**神经衰弱**。

4. **基站天线架设**：在居民楼上设立移动通信基站，天线应尽可能建在楼顶较高的构筑物上或专设的天线塔上。

5. **基站选址避让**：移动通信基站选址宜避开电磁辐射敏感建筑物。在无法避开时，移动通信基站的发射天线水平方向30m范围内，不应有高于发射天线的电磁敏感建筑物。

6. **电磁辐射限值**：我国规定通信基站周围电磁环境控制限值（公众曝露）的功率密度为每平方厘米不超过 **40 μW/cm²**。

7. **施工噪声限值**：通信工程施工期昼间在居民区域的等效声级不得超过 **70dB(A)**。

8. **文物保护**：在文物保护单位周围建设控制地带内的建设工程，不得破坏文物保护单位的环境风貌，其设计方案应征得文化行政管理部门同意。

9. **三同时制度** ★★★：通信工程建设项目中配套的环境保护设施和防治措施，必须执行"三同时制度"，即**与主体工程同时设计、同时施工、同时投产使用**。

10. **四大环境问题**：通信工程中易引发的环境问题归纳为4个方面——**电磁辐射、生态环境、噪声影响、有害物质回收**。

### 二、考试记忆要点

| 考点 | 关键数字/内容 |
|------|--------------|
| 公众曝露功率密度限值 | ≤40 μW/cm² |
| 施工昼间噪声限值 | ≤70 dB(A) |
| 基站天线水平避让距离 | 30m内无高于天线的敏感建筑 |
| 核心制度 | 三同时（同时设计/施工/投产使用） |
| 四大环境问题 | 电磁辐射、生态环境、噪声、有害物质 |

## 必背公式速查

| 公式 | 含义 | 单位 |
|------|------|------|
| $I = -\log_2 P$ | 单个消息信息量 | bit |
| $H(X) = -\sum P(x_i)\log_2 P(x_i)$ | 信源熵 / 平均信息量 | bit/符号 |
| $R_B = 1/T_B$ | 码元速率 | Baud |
| $R_b = R_B \cdot H(X)$ | 信息速率 | b/s |
| $R_b = R_B \cdot \log_2 M$ | 信息速率（等概M进制） | b/s |
| $\eta = R_B/B$ | 频带利用率 | B/Hz |
| $P_e = N_{错}/N_{总}$ | 误码率 | 无 |
| $P_b = B_{错}/B_{总}$ | 误信率 | 无 |
| $P_e \leq P_b$ | 误码率与误信率关系 | — |

## 常见易错点

1. **码元速率 vs 信息速率**：码元速率与进制数无关，信息速率与进制数有关（等概时乘 log₂M）
2. **熵最大条件**：等概分布时信源熵最大
3. **误码率 vs 误信率**：二进制时两者相等，多进制时 $P_e \leq P_b$
4. **有效性与可靠性**：一对矛盾，可互换（如用更宽带宽换取更高可靠性）
5. **模拟vs数字指标**：模拟用带宽/信噪比，数字用速率/误码率
6. **调制 vs 解调**：调制 = 基带→频带（频谱搬移），解调 = 频带→基带
