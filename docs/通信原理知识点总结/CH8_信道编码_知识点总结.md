## (1) 纠错编码的基本概念 ★★★

### 一、核心概念

| 术语 | 定义 |
|------|------|
| **码长n** | 码字中码元总数 |
| **码重W** | 码字中"1"的个数 |
| **码距d（汉明距离）** | 两码字对应位不同的个数 |
| **最小码距d_min** | 所有码字间最小的码距 |
| **编码效率η** | η = k/n = 1 − r/n（k信息位, r监督位） |

### 二、最小码距与纠检错能力 ★★★

| 用途 | 条件 |
|------|------|
| 检测e个错误 | $d_{min} \geq e + 1$ |
| 纠正t个错误 | $d_{min} \geq 2t + 1$ |
| 纠t个同时检e个 (e>t) | $d_{min} \geq t + e + 1$ |

### 三、差错控制方式

| 方式 | 说明 |
|------|------|
| **前向纠错(FEC)** | 发端发纠错码，收端自动纠错 |
| **检错重发(ARQ)** | 发端发检错码，有错请求重发 |
| **混合纠错(HEC)** | FEC + ARQ |

### 四、纠错编码的性能

- 编码后信噪比需求降低（节省功率），或同样信噪比下误码率降低
- 代价：码元增加→带宽增加→噪声增大（需权衡）

## (2) 线性分组码 ★★★

### 一、基本概念

**分组码(n,k)**：k个信息位 + r=n−k个监督位  
**线性码**：监督位是信息位的线性组合（模2）

### 二、生成矩阵G与监督矩阵H

**生成矩阵**：$C = m \cdot G$（m为信息码组）

典型形式：$G = [I_k \mid Q]$（k行n列，系统码）

**监督矩阵**：$H \cdot C^T = O$（零矩阵）

典型形式：$H = [P \mid I_r]$，其中 $P = Q^T$

**典型G与H的关系**：$H = [Q^T \mid I_r]$

### 三、汉明码

满足 $2^r - 1 \geq n$（或 $2^r \geq k + r + 1$）

| 参数 | 公式 |
|------|------|
| 码长 | $n = 2^r - 1$ |
| 信息位 | $k = n - r = 2^r - 1 - r$ |
| 最小码距 | $d_{min} = 3$ |

> 汉明码能纠正1位错误

### 四、伴随式译码

**伴随式**：$S = R \cdot H^T$（或 $S^T = H \cdot R^T$）

- $S = 0$ → 无错（或不可检错误）
- $S \neq 0$ → 有错，$S^T = H \cdot E^T$

**译码流程**：$R \rightarrow S \rightarrow E \rightarrow \hat{C} = R + E$

> 若只有1位错，S等于H中对应错码位置的那一列

### 五、线性分组码的性质

1. **封闭性**：任意两个许用码组之和仍是许用码组
2. **最小码距 = 非零码的最小码重**：$d_{min} = W_{min}$
3. G的各行线性无关，本身就是码字
4. H的各行线性无关

### 六、典型例题（(6,3)码）解题流程

1. 识别(n,k)：G矩阵k行n列
2. 化成典型G：行初等变换 → $[I_k \mid Q]$
3. 求H：$H = [Q^T \mid I_r]$
4. 列全部码组：$C = m \cdot G$
5. 求$d_{min}$：非零码最小码重
6. 列错误图样表：H每一列对应1位错误位置

## (3) 循环码

### 一、基本概念

**定义**：任一码字循环移位后仍是许用码组的线性分组码

**码多项式**：$A(x) = a_{n-1}x^{n-1} + a_{n-2}x^{n-2} + \cdots + a_1 x + a_0$

运算法则：模2运算（加=减=异或）

### 二、生成多项式g(x)

**性质**：
- g(x)是常数项为1、次数为r=n−k的多项式
- g(x)是$(x^n+1)$的一个(n−k)次因式
- 循环码所有码多项式都是g(x)的倍式

**生成矩阵**（非典型形式）：
$$G(x) = \begin{bmatrix} x^{k-1}g(x) \\ x^{k-2}g(x) \\ \vdots \\ x g(x) \\ g(x) \end{bmatrix}$$

### 三、系统循环码编码步骤 ★★★

1. 用 $x^{n-k}$ 乘信息多项式m(x)：$x^{n-k} \cdot m(x)$（信息位左移n−k位）
2. 用g(x)除 $x^{n-k}m(x)$，得余式r(x)（监督位）
3. 码多项式：$A(x) = x^{n-k}m(x) + r(x)$

> 编码结果可被g(x)整除，这是检错依据

**例**：(7,3)码，$g(x)=x^4+x^2+x+1$，信息110 → 1100101

### 四、循环码译码

1. 计算伴随式：$S(x) = R(x) \mod g(x)$
2. $S(x)=0$ → 无错（或不可检错）；$S(x)\neq0$ → 有错
3. 由S(x)确定错误图样E(x)
4. 纠正：$\hat{A}(x) = R(x) + E(x)$

## (4) 常用检错码

### 一、奇偶校验码
**规则**：加1位使码组中"1"个数为偶（或奇）  
**检错能力**：能检奇数个错，不能检偶数个错，不能纠错  
**编码效率**：η = (n−1)/n

### 二、行列奇偶校验码
**规则**：行列二维奇偶校验  
**特点**：能定位错误从而纠错（矩形4角错误除外）

### 三、恒比码
**规则**：码字中1和0保持恒定比例  
**应用**：电报系统，如ARQ中"7取3"恒比码

### 四、正反码
**规则**：信息位r位+监督位r位（监督位=信息位或反码）  
**纠错能力**：长度为10的正反码能纠1位错，检2位以下错

## 必背公式速查

| 公式 | 含义 |
|------|------|
| $d_{min} \geq e+1$ | 检e个错条件 |
| $d_{min} \geq 2t+1$ | 纠t个错条件 |
| $d_{min} \geq t+e+1$ | 纠t检e条件(e>t) |
| $2^r-1 \geq n$ | 汉明码条件 |
| $n=2^r-1,\ k=n-r,\ d_{min}=3$ | 汉明码参数 |
| $C = mG$ | 编码（生成矩阵） |
| $H \cdot C^T = O$ | 校验方程 |
| $S = R \cdot H^T$ | 伴随式计算 |
| $A(x) = x^{n-k}m(x) + r(x)$ | 系统循环码 |
| $S(x) = R(x) \bmod g(x)$ | 循环码伴随式 |

## 常见易错点

1. d_min ≥ e+1 不是 e（检e个错，d_min至少e+1）
2. 线性分组码封闭性（C1+C2仍是码字），d_min = W_min
3. 系统循环码编码：先左移n−k位再除以g(x)取余式
4. 伴随式S=0不等于绝对无错（可能发生不可检错误）
5. $H = [Q^T \mid I_r], G = [I_k \mid Q]$，Q和P互为转置
