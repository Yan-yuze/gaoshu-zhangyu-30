## (1) 二进制数字调制原理 ★★★

### 一、2ASK（振幅键控/OOK）

**表达式**：
$$e_{2ASK}(t) = s(t)\cos\omega_c t,\quad s(t) = \sum a_n g(t-nT_B)$$

**产生**：相乘法 / 键控法  
**波形**：1→有载波，0→无载波

**功率谱**：基带谱搬移到载频两侧（无离散载波分量时）

**带宽**：$B_{2ASK} = 2f_B = 2R_B$（主瓣）

**频带利用率**：$\eta = R_B/B = 0.5$ Baud/Hz

**解调**：包络检波（非相干）/ 相干解调

---

### 二、2FSK（频移键控）

**表达式**：
$$e_{2FSK}(t) = s_1(t)\cos\omega_1 t + s_2(t)\cos\omega_2 t$$

**产生**：模拟调频法（相位连续）/ 键控法（相位可能不连续）

**带宽**：$B_{2FSK} = |f_2 - f_1| + 2f_B$

**频带利用率**：$\eta = R_B / (|f_2-f_1|+2f_B)$

**解调**：
1. 包络检波法（非相干）：两路BPF+包络检波，比较大小判决
2. 相干解调法
3. 过零检测法

---

### 三、2PSK（相移键控）

**表达式**：
$$e_{2PSK}(t) = s(t)\cos\omega_c t,\quad s(t)为双极性NRZ$$

等效：$e_{2PSK}(t) = A\cos(\omega_c t + \varphi_n),\ \varphi_n = 0\ \text{或}\ \pi$

**带宽**：$B_{2PSK} = 2f_B$（同2ASK）

**频带利用率**：$\eta = 0.5$ Baud/Hz

**解调**：必须相干解调

**载波相位模糊问题**（倒π现象）：本地载波如果反相，判决结果全部取反 → 引出了2DPSK

---

### 四、2DPSK（差分相移键控）

**原理**：用相邻码元的相位差表示信息

| 编码规则示例 | $\Delta\varphi = 0$ | $\Delta\varphi = \pi$ |
|-------------|---------------------|----------------------|
| 方式1 | 表示0 | 表示1 |

**产生**：差分编码（$b_n = a_n \oplus b_{n-1}$）+ 2PSK调制

**解调**：
1. **相干解调+码反变换**：先2PSK相干解调恢复$b_n$，再$a_n=b_n\oplus b_{n-1}$
2. **差分相干解调**（延迟解调）：直接比较相邻码元相位差，无需本地载波

> 差分相干解调同时完成解调+码反变换

## (2) 二进制数字调制抗噪声性能 ★★★

设 $r = a^2/(2\sigma_n^2)$ 为解调器输入端信噪比（大信噪比近似公式）

### 误码率公式对比

| 调制方式 | 解调方式 | 误码率Pe（精确） | 大信噪比近似 |
|----------|----------|-----------------|-------------|
| 2ASK | 相干 | $\frac{1}{2}\text{erfc}(\sqrt{r}/2)$ | $\frac{1}{\sqrt{\pi r}}e^{-r/4}$ |
| 2ASK | 非相干 | — | $\frac{1}{2}e^{-r/4}$ |
| 2FSK | 相干 | $\frac{1}{2}\text{erfc}(\sqrt{r/2})$ | $\frac{1}{\sqrt{2\pi r}}e^{-r/2}$ |
| 2FSK | 非相干 | — | $\frac{1}{2}e^{-r/2}$ |
| 2PSK | 相干 | $\frac{1}{2}\text{erfc}(\sqrt{r})$ | $\frac{1}{2\sqrt{\pi r}}e^{-r}$ |
| 2DPSK | 相干 | $\text{erfc}(\sqrt{r})[1-\frac{1}{2}\text{erfc}(\sqrt{r})]$ | $\text{erfc}(\sqrt{r})$ |
| 2DPSK | 差分相干 | $\frac{1}{2}e^{-r}$ | $\frac{1}{2}e^{-r}$ |

### 性能排序（相同r下误码率从小到大）

$$P_{2PSK} < P_{2DPSK} < P_{2FSK} < P_{2ASK}$$

## (3) 性能比较

### 频带宽度

| 调制方式 | 带宽 |
|----------|------|
| 2ASK | $2f_B$ |
| 2PSK | $2f_B$ |
| 2FSK | $\|f_2-f_1\| + 2f_B$ |

### 频带利用率

$$\eta_{2ASK} = \eta_{2PSK} = 0.5，\quad \eta_{2FSK} < 0.5$$

### 对信道特性的敏感性

- **2FSK**：直接比较两路输出，不需设判决门限 → 最不敏感
- **2PSK**：最佳门限=0 → 不敏感
- **2ASK**：最佳门限=a/2，随信号幅度变化 → 最敏感

### 设备复杂度

- 发送端：差不多
- 接收端：相干解调比非相干复杂；2DPSK差分相干最简单

### 应用场景

- 相干2DPSK：高速数据传输
- 非相干2FSK：中低速（衰落信道广泛应用）

## (4) 多进制数字调制（了解）

| 方式 | 带宽 | 特点 |
|------|------|------|
| MASK | $2R_b/\log_2 M$ | 抗噪差 |
| MFSK | $|f_M-f_1|+2f_B$ | 频带利用率低 |
| QPSK/4PSK | $R_b$ | 两路正交2PSK合成 |
| QAM | $R_b/\log_2 M$ | 幅度+相位联合调制，效率最高 |

**QPSK（4PSK）**：$\pi/4$体系或$\pi/2$体系，相邻相位用格雷码

**16QAM**：两路4ASK正交合成，频带利用率比4PSK更高

## 必背公式速查

| 公式 | 含义 |
|------|------|
| $B_{2ASK}=B_{2PSK}=2f_B$ | 2ASK/2PSK带宽 |
| $B_{2FSK}=|f_2-f_1|+2f_B$ | 2FSK带宽 |
| $P_{e,2PSK} = \frac{1}{2}\text{erfc}(\sqrt{r})$ | 2PSK相干误码率 |
| $P_{e,2DPSK} = \frac{1}{2}e^{-r}$ | 2DPSK差分相干误码率 |
| $P_{e,2FSK} = \frac{1}{2}e^{-r/2}$ | 2FSK非相干误码率 |
| $P_{e,2ASK} = \frac{1}{2}e^{-r/4}$ | 2ASK非相干误码率 |
| $b_n = a_n \oplus b_{n-1}$ | 差分编码 |
| $a_n = b_n \oplus b_{n-1}$ | 码反变换 |

## 常见易错点

1. 2PSK载波相位模糊（倒π）→ 使用2DPSK解决
2. 2FSK过零检测法：频率→过零点数→脉冲数→直流分量→判决
3. 同r时2PSK误码率最小（最好），2ASK最大（最差）
4. 2DPSK相干解调约等于2PSK的2倍误码率
5. MASK/MPSK/QAM频带利用率逐级提升
